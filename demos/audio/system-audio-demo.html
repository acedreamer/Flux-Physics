How to Use System Audio:

  1. Select "System Audio" from the source dropdown
  2. Click "Audio Mode" to enable
  3. When prompted, click "Share" in the screen sharing dialog
  4. Make sure "Share system audio" checkbox is checked
  5. Select the screen or application to capture audio from
  6. Play music or audio on your system to see visualization
Note: System audio capture works best in Chrome and Edge browsers. If unavailable, the system will automatically fall back to microphone input.
